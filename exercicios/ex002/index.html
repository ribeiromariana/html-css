Paragrafos e quebras de linha

Voçê pode escrever no paragrafo de qualquer jeito, basta estar entre as tags <p> e </p>

Se precisar quebrar o texto algum lugar especifico,
como esse, pode-se usar a tag <br>


Vamos começar com alguns símbolos:
® = marca registrada
© = copyright
™ = ??
¥ = yen
€ = euro
£ = libra
¢ = centavos de dolar
Δ = Delta que conhecemos ksk
↓ = seta pra baixo
↑ = seta pra cima
← = seta pra esquerda


Agora vamos adicionar alguns emoji:
🍉 = melancia
🥓 = bracon hmm
🥛 = leitinho
